Contenido Plugin and Extension API (PEA)

Welcome to "Contenido Plugin and Extension API" (PEA)...

Contenido PEA is a set of extensions to the content management system 'CONTENIDO' grouped into the main topics:

  • Contenido Plugin and Extension API (PEA) Framework

    a set of classes, functions and scripts for easier and standarized plugin and module development

  • Contenido Plugin and Extension API (PEA) Admin

    a set of classes, functions and standarized applications for plugin and module administration and management
